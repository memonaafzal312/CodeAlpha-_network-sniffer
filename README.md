#  Basic Network Packet Sniffer

A Python-based network packet sniffer that captures and analyzes network traffic in real-time. Built for educational purposes to understand network protocols and packet structure.

##  Features

- Capture live network packets
- Analyze IP, TCP, UDP, and ICMP protocols
- Display source/destination IP addresses
- Show port numbers for TCP/UDP
- Real-time packet monitoring
- Clean console output format

##  Technologies Used

- **Python 3.13
- **Scapy** - Packet manipulation library
- **Socket Programming** - Raw socket handling

# Open Command Prompt as Administrator
python sniffer.py
